## Project Overview
This project is an Oracle PL/SQL-based system designed to manage the collection, tracking, and recycling of electronic waste. It supports environmental sustainability by ensuring traceability of e-waste from citizens to licensed recyclers while rewarding responsible disposal.

## Problem Statement
Improper disposal of electronic waste causes serious environmental and health risks. Many recycling processes lack traceability, accountability, and data-driven decision-making.

## Objectives
- Track e-waste from collection to recycling
- Enforce business rules using PL/SQL
- Reward citizens for responsible disposal
- Provide analytics for environmental decision-making
