# Dashboards - E-Waste Collection & Recycling System

## Executive Summary
- **KPI Cards**:
  - Total Collections
  - Total Items Collected
  - Top Collector
  - Percentage of Compliant Collections
- **Trend Analysis**: Line chart showing weekly collection growth
- **Category Distribution**: Pie chart by item type

## Audit Dashboard
- **Violations Table**: List of collection attempts on restricted days  
- **Denied Actions Chart**: Bar chart showing attempts blocked by triggers  
- **User Activity Heatmap**: Collector activity per location  

## Performance Dashboard
- **Top Collectors**: Ranking by total collections  
- **Item Condition Trend**: Damaged vs functional items collected  
- **Collection Volume by Location**: Map visualization  

## Mockups
- Canva, Lucidchart, or Tableau can be used to create sample dashboards  
- Ensure screenshots of dashboard mockups are added to `screenshots/` folder for GitHub

**Pro Tip:** Use different colors for KPIs to make dashboards visually attractive and easy to interpret.
